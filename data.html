DATA VISUALIZATION:

Data visualization provides an important suite of tools for gaining a qualitative understanding.
For this we have used Matplotlib library. As it is the foundation for many other plotting libraries and plotting support in higher level libraries such as Pandas.

Seaborn :

seaborn is a data visualization library in python based on Matplotlib. It has different functions like count-plot, bar-plot, line-plot etc..
The count-plot() method is used to show the counts of observations in each categorical bin using bars.
The bar-plot() method creates a plot taking a mean per categorical column.
